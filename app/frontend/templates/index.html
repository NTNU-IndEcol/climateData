Climate Data Extraction Tool

Extract and analyze climate data for European countries

About This Tool

This tool is used to extract climate data in Europe. Currently, the available Variables include:

  • 2-meter air temperature (t2m)
  • 10-meter wind speed (ws10)
  • Surface runoff (ro)
  • Total cloud cover (tcc)
  • Surface downward shortwave solar radiation (ssrd)

Data Processing Methods

The ERA5 reanalysis data (available at Copernicus Climate Data Store) is utilized to calculate daily metrics with different Methods including:

  • Daily mean (daymean)
  • Daily maximum (daymax)
  • Daily minimum (daymin)
  • Daily accumulation (daysum)

Data Information

  • Near surface 2m air temperature (t2m): Unit is °C. Data converted from Kelvin to Celsius.
  • Near surface 10m wind speed (ws10): Unit is m/s. Computed as: WS = √(u10² + v10²)
  • Runoff (ro): Unit is meters (m). Only daily accumulation data available.

Time period: 1991-01-01 to 2024-12-31

Percentile indicators: Daily values below/above specific percentiles are marked as 1, otherwise 0. Calculated for bottom percentiles (1-5%) and top percentiles (95-99%).

For country-specific data, use the Natural Earth Vector to extract boundaries. You can extract data for the Whole Country or Power Sub-region.
